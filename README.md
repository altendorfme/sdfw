## Requirements

- Windows 10+
- .NET 10.0 Runtime

## How to work?

- Install and Start or run `SDfW.UI.exe` on portable version

## Files

Configuration:
```
C:\ProgramData\Sdfw\config.json
```
Logs:
```
C:\ProgramData\Sdfw\Logs\
```
